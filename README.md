## 📝 功能

- 新增支出記錄
- 支出圖表分析（柱狀圖）
- 自訂花費類別
- 可識別裝置深色/淺色模式並轉換
- 可儲存並回復誤刪的支出
- 多平台支援：Android / iOS

## 🔧 技術使用

- Flutter
- Dart
- 自製 Chart 元件
- Local state 管理（無使用第三方套件）

## 📦 如何使用

```bash
git clone https://github.com/Bruce-159/flutter-expense-tracker.git
cd flutter-expense-tracker
flutter pub get
flutter run
